/v2/client/{clientId}/customaccounttag

METHOD: GET

Retrieves the account's custom tag information.

Parameters:

Name Location Required Description Default Format Example
clientId path yes The client id - number 18901
accessTokenKey header yes Security parameter - string 63129107067A520E6B7DE18D5E1CB2BF
accessTokenSecret header yes Security parameter - string 072639740ac239333e3708a1623b9de3590d972e6b1060071b726ecfd9b21600

 

Response:

Content-Type: application/json

Status Code Reason
401 AccessTokenKey and AccessTokenSecret are wrong or invalid.
403 You do not have permission to perform the requested operation.
403 The client id {clientId} is inactive.
403 You have reached the maximum number of calls for this api.
404 The client {clientId} doesn't exist.
