/v2/client/{clientId}/event/{eventId}/forgetall

METHOD: POST

Forgets personal data for all registrant records in the specified event.

Parameters:

Name Located in Required Description Default Schema Example
clientId path yes The client id - integer 18091
eventId path yes The event id - integer 1487544
accept header no Output format. possible values: Application/xml or Application/json application/json string Application/json
accessTokenKey header yes client access token key - string 63129107067A520E6B7DE18D5E1CB2BF

“multipart/formdata"		 - string "application/x-www-form-urlencoded"

Content-Type: application/json, application/xml, text/plain
