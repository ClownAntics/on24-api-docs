/v2/client/{clientId}/event/{eventId}/attendeesession

METHOD: GET

Returns a list of attendee viewing sessions for a particular event. All dates and times are in Pacific Time.

Parameters:

Name Located In Required Description Default Schema Example
clientId path yes The client id - number 1498
eventId path yes The event id - number 1487542
accessTokenKey header yes Security Parameter - string 63129107067A520E6B7DE18D5E1CB2BF
accessTokenSecret header yes Security Parameter - string 072639740ac239333e3708a1623b9de3590d972e6b1060071b726ecfd9b216001b45391717d9fbeb780438
startDate query no Limit results to sessions created from startDate - string (date) 2021-04-10
endDate query no Limit results to sessions with last update until endDate Current date if startDate value was provided and endDate value was not provided string (date) 2021-04-15
sessionType query no Filter by session type All string ondemand
pageoffset query no Number of page to retrieve 0 number 5
itemsPerPage query query Number of items to be retrieved per page 100 number 25

 

Notes:

  • sessionType supported values:
    • "all" (default)
    • "live"
    • "ondemand"
    • "blocked"

 

Response:

Content-Type: application/json

Status Code Reason
200 Return list of attendee viewing sessions for an event
401 AccessTokenKey and AccessTokenSecret are wrong or invalid.
403 You do not have permission to perform the requested operation.
403 You have reached the maximum number of calls for this api.
403 The client id {clientId} is inactive.
403 The event {eventId} is inactive.
404 The event {eventId} doesn't exist.
