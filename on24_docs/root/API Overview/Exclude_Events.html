Exclude Events

A tag named "APIEXCLUDE" can be added to ON24 events to exclude the event, registrant and attendee data from flowing through the Client Level endpoint responses. The tag allows excluding specific events from being returned in the API responses and flowing through any integration service leveraging the API.

Tag:

  • APIEXCLUDE

Notes:

  • Tag name is not case-sensitive, so APIEXCLUDE = APIExclude = apiexclude, etc.
  • The tag exclusion only applies to the Client Level endpoints. Calling out to a specific event using the Event Level endpoints with the eventid in the request path will still return data.
