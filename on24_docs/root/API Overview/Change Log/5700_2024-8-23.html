Change Log

[VERSION 57.0.0] - 2024-08-23

Added

  • Event Management > Create a Webinar: Added support to create Forums.
  • Analytics > Client Level > Events: Added the campaigncode field to the events in the response.
  • All attendee endpoints:
    • Added new reactions array for webinar attendees in the response.
    • Added new shareidea array for webinar attendees in the response.

Changed

  • Analytics > Event Level > Metadata and Usage: Only for Go Live events, the old eventlocation field in the response was replaced by the new campaigncode field.
