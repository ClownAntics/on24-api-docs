Poll Deletion or Modification

Users should not delete or change poll options after there are already responses from the attendees. See below for more information.

  • If a poll is deleted, it is a hard deletion from our database.
  • If deletion happens after there has been responses from the audience, the poll responses will no longer be available via the API.
  • If the poll is a multiple choice question, and the question options are changed or deleted, this will affect any previous responses to the poll. For example, if an attendee has responded by selecting the first option of the poll, and the first option is subsequently deleted, we cannot accurately report the response of that attendee.
