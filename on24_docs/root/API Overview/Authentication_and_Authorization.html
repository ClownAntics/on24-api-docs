Authentication and Authorization

In Elite, go to Analytics > API Dashboard. Click on the Provision a New API Access Token button to generate the following:

  • Client ID
  • Access Token Key
  • Access Token Secret

 

Authentication for API requests is handled via custom HTTP request headers:

  • accessTokenKey – The client access token key
  • accessTokenSecret – The client access token secret

 

The Client ID value is included in the path of the API request URL.

 

Response error codes:

400 The client {clientId} doesn't exist.
401 AccessTokenKey and AccessTokenSecret are wrong or invalid.
401 AccessTokenKey and AccessTokenSecret are deactivated.
401 You do not have permission to perform the requested operation. 
401 The admin id {adminId} is inactive.
403 The client id {clientId} is inactive.
403 Client contract with id {clientId} is expired.
