Obtaining API Access Tokens

Login to the ON24 platform to generate the Token Key, Token Secret, and Client ID for use with the ON24 REST API.

 

  1. Log In to ON24
  2. Go to Platform Tools > Integrations > API Tokens
  3. Click on the Provision a New API Access Token button. This will generate the Token Key and Token Secret, as well as show the Client ID for use with the API. If you see a delay in the credentials displaying, please try refreshing the page.

 

 

API Access Levels

ON24 provides granular endpoint access per token, and every token can have multiple access levels or have full access to all endpoints.

Access Level Options:

  • All endpoints
  • Event Management endpoints
    • Create a Webinar
    • Copy a Webinar
    • Update a Webinar
    • Delete a Webinar
    • Update Email Notification
    • Update Email Calendar Reminder
  • Account-level Analytics endpoints
    • Event Listing
    • Engaged Accounts
    • Sub-Clients
    • Client-Level Registrants
    • Registrant (by email)
    • Leads
    • Client-Level Attendees
    • Attendee (by email)
    • Presenters
    • User Questions
    • PEP
  • Event-level Analytics endpoints
    • Metadata & Usage
    • Registrants
    • Attendees
    • Resources Viewed
    • Surveys
    • Polls
    • Attendee Viewing Sessions
    • Presenters
    • Email Notification Details
    • Email Calendar Reminder
    • Engagement Hub Content Listing
  • Registration endpoints
    • REST Registration
    • Forget Registrant (Client-Level)
    • Forget All Registrant (Client-Level)
    • Forget All Registrant (Event-Level)
    • Update Registrant (Client-Level)
  • Helper endpoints
    • Retrieve Language Codes
    • Retrieve Timezones Codes
    • Retrieve Event Types
    • Retrieve Signal Contacts
    • Retrieve Event Profiles
    • Retrieve Event Managers
    • Retrieve Account Managers
    • Retrieve Sales Rep Contacts
    • Retrieve Technical Rep Contacts
    • Retrieve Registration Fields
    • Replacement Tokens
