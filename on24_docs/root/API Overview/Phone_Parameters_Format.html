  • ››Phone Parameters Format

We validate the phone input parameters according to the following rules:

  • Should contain at least 7 digits
  • Non-numeric characters are removed
  • Phone numbers with 7 digits are formatted as ###-#### in the response
  • Phone numbers with 10 digits are formatted as (###)-###-#### in the response
  • Max allowed length: home_phone=1000|work_phone=1000|fax=1000 (if length is greater than 1000 it is trucanted
