/v2/client/{clientId}/event/{eventid}/speakerbio

METHOD: POST

Creates a new speaker bio element in the Speaker Bio widget of a webinar.

Parameters:

Name Located in Required Description Default Schema Example
clientId path yes The client id - string 18091
eventId path yes The event id - string 13658130
accessTokenKey header yes client access token key - string 63129107067A520E6B7DE18D5E1CB2BF
accessTokenSecret header yes client access token secret - string 072639740ac239333e3708a1623b9de3590d972e6b1060071b726ecfd9b216001b45391717d9fbeb780438
content-type header yes It must be: “multipart/form-data" - string "multipart/form-data"
file formData no Speaker bio image - File  
metadata formData yes JSON body with speaker data - JSON {
   "name":"John Doe",
   "title":"Product Manager",
   "company":"ON24",
   "email":"john.doe@on24.com",
   "twitter":"https://x",
   "linkedin":"https://x",
   "website":"https://x",
   "facebook":"https://x",
   "description":"Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."
}

 

Notes:

  • A Speaker Bio widget must already exist in the webinar. If a Speaker Bio widget does not exist, a 404 response will be returned.

 

Supported File Types for Speaker Bio Image:

  • .jpg
  • .gif
  • .png

 

Required 'metadata' JSON key-value pairs:

  • name
  • email

 

'metadata' restrictions:

  • Only up to three (3) among "email", "twitter", "linkedin", "website", and "facebook" can be provided. Providing more than three will result in a 400 Invalid metadata response.

 

Response:

Content-Type: application/json

Status Code Reason
200 Speaker Bio created.
400 Invalid metadata or file is not accepted filetype.
401 AccessTokenKey and AccessTokenSecret are wrong or invalid.
403 You do not have permission to perform the requested operation.
403 You have reached the maximum number of calls for this api.
403 The client id {clientId} is inactive.
403 The event {eventId} is inactive.
404 The client {clientId} doesn't exist. 
404 The event {eventId} doesn't exist.
404 The Speaker Bio widget is not available for this event.
