/v2/client/{clientId}/event/{eventId}

METHOD: DELETE

Deletes a webinar.

Parameters:

Name Located In Required Description Default Schema Example
clientId path yes The client id - number 18091
eventId path yes The event id - number 9486514
accept header no Output format. possible values: Application/xml or Application/json application/json string application/json
accessTokenKey header yes client access token key - string 63129107067A520E6B7DE18D5E1CB2BF
accessTokenSecret header yes client access token secret - string 072639740ac239333e3708a1623b9de3590d972e6b1060071b726ecfd9b216001b45391717d9fbeb780438

 

Response:

Content-Type: application/json, application/xml, text/plain

Status Code Reason
200 Event Deleted.
401 The access token secret or access token key was not found
403

You don't have access to this client id; or

You don't have ability to delete this event; or

You reached the maximum number of calls for this endpoint
404 Event not found for this client or its children
410 Event has already been deleted
412 Event does not meet criteria for deletion
