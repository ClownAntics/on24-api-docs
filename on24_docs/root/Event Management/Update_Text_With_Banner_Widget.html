/v2/client/{clientId}/event/{eventid}/textwithbanner

METHOD: POST

Updates the text content of a webinar's Text With Banner widget.

Parameters:

Name Located in Required Description Default Schema Example

content-type header yes It must be: “application/x-www-form-urlencoded" - string "application/x-www-form-urlencoded"
metadata formData yes JSON object containing all the data needed to update the widget - JSON

Notes:

  • At least one Text With Banner widget must already exist in the webinar. If a Text With Banner widget does not exist, a 404 response will be returned.
  • The "matchname" metadata field value must be the same as the Text With Banner widget name. If a Text With Banner widget does not exist with the "matchname" value, a 404 response will be returned.
  • After successfully executing this endpoint, the previously existing text content of the widget will be overwritten.
  • type parameter options:
    • "text" is intended for plain text content.
    • "table" is intended for tabular information. All rows must have the same number of columns and these must be called column1 to columnN, otherwise a 400 response will be returned.
