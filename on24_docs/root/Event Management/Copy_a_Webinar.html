/v2/client/{clientId}/event?eventsource={eventid}

METHOD: POST

Create a copy of an existing Webinar.

Notes:

  • When an existing event id is provided in query parameter eventsource, the new event created by this Endpoint will copy all the other fields from this existing event except the liveStart and liveDuration. 
  • Any of the parameters can be overridden, all parameters in the Event Creation are accepted.
  • Required form parameters: liveDuration and liveStart.
  • The copy process will copy the event source's tags (funnel, campaign, sales) and resources uploaded to it (PDF,mp4, etc)
  • liveDuration parameter conditions:
    • When copying a webinar, users are able to provide a liveDuration value in 15 minute increments (e.g. 15, 30, 45, 60, etc.) up to 120 minutes.
    • Above 120 minutes, increments will be 30 minutes up to 480 minutes.
  • liveStart accepted date/time formats:
    • MM/DD/YYYY hh:mm a (08/26/2016 09:00 AM)
    • YYYY-MM-DDThh:mm:ss Default timezone will be Pacific Time. (2016-08-26T09:00:00) 
    • YYYY-MM-DDThh:mm:ss±zz:zz where ±zz:zz is timezone's offset (2016-08-26T09:00:00-07:00)
    • JavaScript Date.toString() format is also supported (Wed Apr 04 2018 16:12:41 GMT+0200 (CEST) )
  • tagCampaign parameter conditions:
    • When copying a webinar users are allowed to add up to 10 custom tags via the tagCampaign parameter.
    • Every tag that is associated with the webinar should be available in the tags array field in the response.
    • The following characters can not be used and will result in an error: £$!%&*()}{@#~?><',"|=_+¬
  • customAccountTag parameter conditions:
    • Supported only for client accounts with enabled custom account tags.
    • When copying a webinar for an account with enabled custom account tags, users are allowed to add up to 3 tag IDs
    • Provided values must match configured client account values.
  • languageCd / countryCd parameters conditions:
    • Following are allowed pairs of language/country values when copying an event:
      • Chinese (Simplified): languageCd="zh", countryCd="CN"
      • Chinese (Traditional): languageCd="zh", countryCd="TW"
      • English (UK): languageCd="en", countryCd="UK"
    • For other languages, No countryCd is allowed.
  • When copying a webinar, the following features will be copied from the source event unless their values are overriden:
    • Title. In the following format: 'Copy of' + [source event name]
    • Event Type
    • Duration
    • Language
    • Funnel Tags
    • Campaign Tags
    • Custom Account Tags
    • Registration page:
      • Banner
      • Registration fields
      • Page design
    • Email notifications configuration
    • Videos and Slides
    • Console customization
  • Elements that are not copied:
    • Start time
    • Archive configuration
    • Security configuration
